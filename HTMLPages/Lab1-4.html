Common HTML Tags

These are the most commonly used HTML tags and a short description of each.

Block Elements

Block elements take up their own line and take up the full width of the page.

body

This tag defines the document's body.

head

Defines information about the document.

html

Defines the root of an HTML document.

div

Defines a section in a document.

section

Defines a section in a document.

header

Defines a header for a document or section.

main

Specifies the main content of a document.

article

Defines an article.

nav

Defines navigation links.

footer

Defines a footer for a document or section.

p

Defines a paragraph.

aside

Defines content aside from the page content.

li

Defines a list item.

Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

a

This tag defines a hyperlink.

strong

Defines important text.

em

Defines emphasized text.

img

Defines an image.

span

Defines a section in a document.
